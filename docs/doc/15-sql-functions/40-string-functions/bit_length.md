title: BIT_LENGTH
---

Return the length of a string in bits.

## Syntax

```sql
BIT_LENGTH(str)
```

## Arguments

| Arguments   | Description |
| ----------- | ----------- |
| str         | The string. |

## Return Type

A number data type value.

## Examples

```sql
SELECT BIT_LENGTH('Word');
+----------------------------+
| SELECT BIT_LENGTH('Word'); |
+----------------------------+
| 32                         |
+----------------------------+
```
