---
title: REVERSE
---

Returns the string str with the order of the characters reversed.

## Syntax

```sql
REVERSE(s)
```

## Arguments

| Arguments   | Description |
| ----------- | ----------- |
| s | The string value. |

## Return Type

A String data type value.

## Examples

```sql
SELECT REVERSE('abc');
+----------------+
| REVERSE('abc') |
+----------------+
| cba            |
+----------------+
```
