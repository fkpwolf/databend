---
title: VERSION
---

Return the current version information of DatabendQuery.

## Syntax

```
SELECT VERSION()
```

## Examples

```sql
SELECT VERSION();
+----------------------------------------------------------------------------------------+
| version()                                                                              |
+----------------------------------------------------------------------------------------+
| DatabendQuery v-0.1.0-0f9ec31-simd(1.56.0-nightly-2021-08-10T15:25:36.875868571+00:00) |
+----------------------------------------------------------------------------------------+
```
