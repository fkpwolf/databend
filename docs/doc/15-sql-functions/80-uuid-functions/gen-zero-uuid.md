---
title: GEN_ZERO_UUID
---

Generate zero UUID.

## Syntax

```sql
GEN_ZERO_UUID()
```

## Return Type

Zero UUID.

## Examples

```sql
SELECT gen_zero_uuid();
`+--------------------------------------+
| gen_zero_uuid()                      |
+--------------------------------------+
| 00000000-0000-0000-0000-000000000000 |
+--------------------------------------+
```
