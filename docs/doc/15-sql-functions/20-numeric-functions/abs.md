---
title: ABS
description: ABS(x) function
---

Returns the absolute value of x. 

## Syntax

```sql
ABS(x)
```

## Examples

```sql
SELECT ABS(-5);
+---------+
| ABS(-5) |
+---------+
|       5 |
+---------+

SELECT ABS(-5.5);
+-----------+
| ABS(-5.5) |
+-----------+
|       5.5 |
+-----------+
```
